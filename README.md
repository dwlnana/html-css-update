## HTML and CSS Project (Improved Version)

### A fully responsive project built using HTML and CSS

**Live Site:** [View the website](https://dwlnana.github.io/html-css-update/)

This repository contains an improved version of my original HTML and CSS Rainy Days project. The project was updated to refine the layout, improve styling, and clean up the code structure.

The goal of this project was to transform the Rainy Days design (from the Design Course) into a fully responsive website using only HTML and CSS.

### Features

- Multi-page website with navigation between pages
- Layout built using Flexbox and CSS Grid
- Improved styling and structure compared to the original project
- Responsive design for different screen sizes

### Technologies

- HTML
- CSS
